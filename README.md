# nodevagrant

## Features

* Apt-fast - much faster apt-get installs
* Samba - folder share via network
  * devs folder for windows users
* Virtualbox shared folder
  * dev folder for unix users
* Node
  * NVM
* Mongo
* Docker
  * Docker-compose
  * Docker Remote API
* PhantomJS
* Utility shellscripts
* Command line tools (git, mc, htop, etc.)
